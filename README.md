# BlacklabelOps/DockerDev

Vagrantfile for a Docker development environment.

## Build and Use Box

~~~~
$ vagrant up
$ vagrant ssh
~~~~    

> This will start and publish the Vagrant box and connect to the running box.

## What's inside?

* CentOS7
* Docker Latest
* Docker-Compose

## Tested with

* Vagrant 1.7.2, 1.7.1
